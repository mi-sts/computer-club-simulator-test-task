Тестовое задание выполнил Мищенко Станислав.
Запуск программы производится путём вызова команды: $ python3 task.py test_file.txt
Файл указывается первым аргументом при запуске программы (в данном случае test_file.txt).

В директории tests/algorithm/ находятся папки вида test_*, содержащие тестовые данные для программы в файле input.txt и ожидаемый результат работы программы в файле output.txt.

В файле tests/test_time.py содержатся юнит-тесты, реализованные с использованием библиотеки pytest.
